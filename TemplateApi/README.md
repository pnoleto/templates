
# TemplateApi
Template de WebApi construido com padrões altamente utilizados no meio da programação comercial.

Oque tem nesse template?

* Domain Driven Design
* Clean Architecture
* Entity Framework
* AutoMapper
* MadiatR
* Swagger
* OpenTelemetry
* Api Versioning
* RESTful
* SQL Server Support
* PostgreSQL Support
* InMemoryDb Support
* HangFire Recurring Jobs
* TAP Pattern
* Extension Methods
